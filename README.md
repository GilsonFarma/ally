# Site Psicóloga Premium

Projeto em React + Vite criado para você editar e publicar.

## Como rodar

1. Abra a pasta no VS Code
2. Rode no terminal:

```bash
npm install
npm run dev
```

## Como gerar versão final

```bash
npm run build
```

## Arquivos para trocar primeiro

- `public/logo-placeholder.svg` → troque pela logo real
- `public/hero-placeholder.svg` → troque pela foto real
- `src/App.jsx` → altere textos, WhatsApp, Instagram e e-mail
- `src/styles.css` → ajuste detalhes finos de cor, espaçamento e tipografia

## Observações

- Usei `Playfair Display` como substituta visual para títulos
- Usei `Mulish` para textos corridos
- O projeto já está responsivo
- Não inclui seção de valores, queixas ou traumas
